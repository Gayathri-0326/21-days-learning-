2. Conditional Statements in python:
   Conditional Statements are used to perform differrefnt actions based on different conditions
Types:
*if
*if-else
*if-elif-else
i) if statement - Executes code only if condition is true
ii)if-else statement - Executes one block if condition is true and another if false
iii)if-elif-else statement - Used to check multiple conditions
