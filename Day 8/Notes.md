Tuples in Python:
A tuple is a collection of items similar to a list but it is immutable (cannot be changed).
Tuples are ordered and allow duplicate values.
Syntax:
tuple_name = (item1, item2, item3)
Example:
numbers = (10, 20, 30)
print(numbers)
Output:
(10, 20, 30)

Accessing elements:
numbers = (10, 20, 30)
print(numbers[0])
print(numbers[1])
Output:
10
20

Tuple with different data types:
data = ("Gayathri", 20, True)
print(data)
('Gayathri', 20, True)

Length of tuple:
numbers = (1, 2, 3, 4)
print(len(numbers))

Output:
4

Loop through tuple:
numbers = (1, 2, 3)
for num in numbers:
    print(num)

Output:
1
2
3

Difference between List and Tuple:
List → mutable (can change)
Tuple → immutable (cannot change)
