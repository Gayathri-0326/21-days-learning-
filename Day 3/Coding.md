Type casting:
#Convert float into int:
x=10.5
print(int(x))
#Convert int to float:
y=5
print(float(y))
#Convert int to String:
z=100
print(str(z))
#Convert number to boolean:
a=1
print(bool(a))
Output:
10
5.0
100
True
Operators in Python:
1.Arithmetic Operators
a=35
b=5
print(a+b)
print(a-b)
print(a*b)
print(a/b)
print(a%b)
Output:
40
30
175
7
0
2.Comparison Operators:
print(4>3)
print(4==3)
Output:
True
False
3.Logical Operators:
print(True and False)
print(True or False)
Output:
False
True
